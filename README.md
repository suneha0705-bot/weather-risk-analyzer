# Weather Risk Analyzer

A Python-based system that fetches live weather data and converts it into
risk levels using rule-based decision logic.

## Features
- Live weather data using OpenWeather API
- Multi-city analysis
- Rule-based risk engine
- Pandas tabular output
- Clean modular Python design

## Tech Stack
- Python
- Requests
- Pandas
- API Integration
- Git & GitHub

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Set environment variable:
   OPENWEATHER_API_KEY=your_api_key
4. Run:
   python main.py
